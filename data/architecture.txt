La aplicación sigue una arquitectura basada en el stack MERN, compuesto por MongoDB, Express, React y Node.js.

El frontend está desarrollado con React, utilizando componentes reutilizables y gestionando el estado de forma modular.

El backend está construido con Node.js y Express, siguiendo un patrón MVC que separa controladores, modelos y rutas.

La base de datos utiliza MongoDB Atlas, permitiendo almacenar datos en formato documento con flexibilidad para adaptarse a cambios en los requisitos.

La comunicación entre frontend y backend se realiza mediante APIs REST.
