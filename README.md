# Sports RAG Assistant

AI assistant basado en RAG (Retrieval-Augmented Generation) diseñado para responder preguntas sobre documentación técnica de un sistema de gestión deportiva.

## Objetivo

Explorar cómo construir un sistema de recuperación semántica que permita consultar información de forma eficiente a partir de documentación estructurada.

El sistema utiliza como caso de estudio la documentación de una aplicación real (gestión deportiva universitaria), aunque está diseñado para ser extensible a otros dominios.

## Stack

- Python
- OpenAI API
- Embeddings
- RAG básico

## Funcionalidades (en progreso)

- Ingesta de documentos
- Búsqueda semántica
- Generación de respuestas con contexto

## Variables de entorno

- OPENAI_API_KEY: tu clave de OpenAI
